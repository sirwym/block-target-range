# 文档索引

《我的方块靶场》是一个基于 Babylon.js、Vite 和 Node test 的第一人称方块靶场 demo。项目目前以浏览器试玩、轻量测试和资源链路验证为主。

## 常用命令

- 启动开发服务器：`npm run dev`
- 运行测试：`npm run test`
- 构建产物：`npm run build`
- 完整检查：`npm run check`
